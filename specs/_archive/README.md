# Archived specs

Closed-chapter specs preserved here for historical context.
Each spec documents a build phase that has shipped and is no longer the source of truth for current behavior — that role belongs to ADR 0016, `CONTEXT.md`, and the per-package `README.md`s.

These files are exempt from the repo-wide rule that bans the `v2` token in `specs/` (issue #437): they describe the original `v1` → `v2` greenfield rebuild that ended in PR #421's cutover and the two follow-up cleanups (#422 — code-only suffix drop, #437 — persistence + JSDoc + HTML-IDs).

Touching a spec here means re-opening a closed chapter; prefer adding a new spec under `specs/` for new work.
